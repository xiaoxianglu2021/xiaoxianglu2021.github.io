## Tmux 是什么？

Tmux 是一个终端复用器（terminal multiplexer），主要是为了将窗口和会话解绑，使得窗口和会话为多对多的关系；

## 安装

```bash
# Ubuntu 或 Debian
$ sudo apt-get install tmux

# CentOS 或 Fedora
$ sudo yum install tmux

# Mac
$ brew install tmux
```

## 使用

```bash
#进入窗口
tmux
#退出窗口
exit
```

快捷键：

```tex
Tmux 窗口有大量的快捷键。所有快捷键都要通过前缀键唤起。默认的前缀键是`Ctrl+b`，即先按下`Ctrl+b`，快捷键才会生效。举例来说，帮助命令的快捷键是`Ctrl+b ?`。它的用法是，在 Tmux 窗口中，先按下`Ctrl+b`，再按下`?`，就会显示帮助信息。

然后，按下 ESC 键或`q`键，就可以退出帮助。
```

### 会话管理

1. 新建会话

   ```bash
   tmux new -s <session-name>		#一个会话会有编码和名称
   ```

2. 重命名会话

   ```bash
   tmux rename-session -t 0 <new-name>
   ```

3. 会话分离	
   命令执行后，就会退出当前 Tmux 窗口，但是会话和里面的进程仍然在后台运行。

   ```bash
   tmux detach
   ```

4. 接入会话

   ```bash
   #使用会话编码
   tmux attach -t 0
   #使用会话名称
   tmux attach -t <session-name>
   ```

5. 切换会话

   ```bash
   tmux switch -t 0
   tmux switch -t <session-name>
   ```

6. 终结会话

   ```bash
   tmux kill-session -t 0
   tmux kill-session -t <session-name>
   ```

7. 会话快捷键

   ```bash
   Ctrl+b d：分离当前会话。
   Ctrl+b s：列出所有会话。
   Ctrl+b $：重命名当前会话。
   ```

### 窗口管理

`tmux split-window`命令用来划分窗格

```bash
# 划分上下两个窗格
tmux split-window

# 划分左右两个窗格
tmux split-window -h
```
